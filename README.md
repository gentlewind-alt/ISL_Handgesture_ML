# Hand Gesture Recognition

This project focuses on recognizing hand gestures using machine learning techniques. The goal is to create a system that can accurately identify different hand gestures in real-time.

## Introduction
Hand gesture recognition is an important aspect of human-computer interaction. This project aims to develop a robust and efficient hand gesture recognition system using state-of-the-art machine learning algorithms.

## Installation
To get started with this project, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/hand_gesture_recognition.git
    ```
2. Navigate to the project directory:
    ```sh
    cd hand_gesture_recognition
    ```
3. Install the required dependencies:
    ```sh
    pip install -r requirements.txt
    ```

4. Download the trained model from Google Drive and place it in the **same directory as the Jupyter Notebook (.ipynb) file**.

## Usage
To use the hand gesture recognition system, follow these steps:

1. Download the trained model from the Google Drive link provided with this project.
2. Place the downloaded model file in the **same directory as the notebook (.ipynb)**.
3. Open the notebook and **run all the cells**.
4. Finally, **run the last cell of the notebook to start the gesture recognition system**.
